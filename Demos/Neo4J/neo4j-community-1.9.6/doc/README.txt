Neo4j Included Documentation
=========================================

Neo4j included documentation, providing reference material
and tutorials.

In addition to topical documents, subdirectories include:

* [javadoc](java/api/index.html) - Javadocs for the Neo4j Libraries
* [manual](manual/text/neo4j-manual.txt) - Neo4j Manual in pure text format
* [manual](manual/html/index.html) - Neo4j Manual in navigable HTML format
* [manual](http://docs.neo4j.org/chunked/1.9.6/) - Neo4j Manual in navigable HTML format (Online w comments)

For a PDF version of the Neo4j Manual, see http://docs.neo4j.org/
